Introduction – Why are AI agents exploding in 2025/2026?

Level ★☆☆☆☆

A modern AI agent is no longer just a chatbot. It is an autonomous system that:

  • reasons step by step
  • uses tools (API, calculator, web search, code execution…)
  • maintains long-term memory
  • collaborates with other agents
  • makes complex multi-step decisions

Most used frameworks today:

  • LangGraph (recommended by LangChain for production)
  • CrewAI (very simple for role-based multi-agents)
  • AutoGen (Microsoft – conversations between agents)
  • OpenAI Swarm / Agents (lightweight and managed)

Key resources to get started:

  • → Official LangGraph website
  • → Free Intro to LangGraph course (LangChain Academy)
  • → Official CrewAI website

Step 1 – First ultra-basic ReAct agent

Level ★★☆☆☆

The ReAct pattern = Reason (think) + Act (act) + observe the result → loop until final answer.

from openai import OpenAI
client = OpenAI()

def get_weather(city: str) -> str:
    return f"{city}: 24°C, sunny (simulation 2026)"

tools = [{
    "type": "function",
    "function": {
        "name": "get_weather",
        "description": "Get the weather for a city",
        "parameters": {
            "type": "object",
            "properties": {"city": {"type": "string"}},
            "required": ["city"]
        }
    }
}]

question = "What's the weather in Osasco today?"
messages = [{"role": "user", "content": question}]

response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=messages,
    tools=tools
)
# ← Real ReAct loop starts here (see step 2)
print(response.choices[0].message.tool_calls)

Step 2 – Complete loop + multiple tools

Level ★★☆☆☆ → ★★★☆☆

Add a while loop, multiple tools, and error handling.

# Simplified ReAct loop excerpt
while True:
    resp = client.chat.completions.create(...)
    if resp.choices[0].message.tool_calls:
        for call in resp.choices[0].message.tool_calls:
            if call.function.name == "get_weather":
                args = json.loads(call.function.arguments)
                result = get_weather(**args)
                messages.append({"role": "tool", "content": result, "tool_call_id": call.id})
    else:
        final = resp.choices[0].message.content
        break

Step 3 – Adding persistent memory

Level ★★★☆☆

Simple → advanced options:

  • Python list (history in messages)
  • ConversationBufferMemory (LangChain)
  • Vector store + RAG for very long memory

Quick LangChain example:

from langchain.memory import ConversationBufferMemory
memory = ConversationBufferMemory(return_messages=True)
# Then inject it into the agent

Step 4 – Modern control with LangGraph

Level ★★★★☆

LangGraph replaces the while loop with a state graph → branches, cycles, human-in-the-loop, persistence, easy debugging.

Minimal example:

from langgraph.graph import StateGraph, START, END

graph = StateGraph(...)
graph.add_node("agent", call_llm)
graph.add_node("tools", execute_tools)
graph.add_conditional_edges("agent", should_continue)
graph.add_edge("tools", "agent")
graph.add_edge(START, "agent")
app = graph.compile()
